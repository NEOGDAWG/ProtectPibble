# Pet Images

This folder contains images representing different health states of the pibble pet.

## Image Files

- `pibble.png` (or `.jpg`, `.svg`, etc.) - Healthiest state (100% health)
- `pibble2.png` - High health state
- `pibble3.png` - Medium health state
- `pibble4.png` - Low health state
- `pibble5.png` - Lowest health state (critical)

## Usage

These images will be used in the GroupDashboardPage to display the pet's current health state.
The image selection will be based on the pet's health percentage.

## Note

Please add the actual image files to this directory with the names listed above.
